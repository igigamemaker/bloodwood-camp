# Bloodwood Camp — PWA

Ta wersja działa jako Progressive Web App.

## Jak wrzucić na GitHub Pages

1. Wgraj do repozytorium pliki z tego ZIP-a:
   - `index.html`
   - `manifest.webmanifest`
   - `service-worker.js`
   - folder `assets/`
2. W GitHub ustaw: Settings → Pages → Deploy from branch → `main` → `/ (root)`.
3. Po zielonym `pages build and deployment` otwórz stronę gry.

## Jak gracz instaluje

### Windows / macOS przez Chrome albo Edge
Po wejściu na stronę gry kliknij ikonę instalacji w pasku adresu albo menu przeglądarki → Install app.

### iPhone / iPad
Safari → Udostępnij → Do ekranu początkowego.

### Android
Chrome → menu ⋮ → Dodaj do ekranu głównego / Zainstaluj aplikację.

Uwaga: to nie wymaga Electron, `.exe`, `.dmg`, podpisu Apple ani GitHub Releases.
